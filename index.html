The Basics of Web and HTML

How the Web Works

 The web is a bunch of computers that communicate with each other. There are 3 main parts:
1. Web Browser
2. Internet
3. Servers

When on the web, the web browser sends requests to a server to display documents on the user's screen. This request is called HTTP.

HTML

 HTML stands for Hypertext Markup Language. Most of the documents on the web are HTML based. You can use HTML to link pages together with hyperlinks.

Tags and Elements

 HTML documents are made up of elements. Tags are used to denote the beginning and end of elements.

Computers are 'Stupid'

 They can only do exactly what you tell them.

Inline vs. Block Elements

 HTML elements are either inline, or on the same like with the text, or block, seperated from the text by an invisible box.
